Project Overview

Autonomous Swarm UAV Systems [Otonom Suru IHA Sistemleri (OSIS)] is a proprietary R&D project funded partly by TUBITAK and conducted by MilSOFT.

I worked as a Motion Planning/Simulation/System Integration engineer from the beginning of the project and played a crucial role in introducing Robotics know-how to the company.

Key Contributions

  • HITL Stack: Created DJI M100 HITL stack to test the developed application(s) in Gazebo environment, with pre-made scenarios dedicated to swarm settings.
  • Cross-Compilation: Created a cross-compilation suite for ROS1 to ARM architecture.
  • Mesh Networking: Created a high-frequency ad-hoc mesh network using only on-board WiFi of Raspberry Pis.

Note: Since this is a proprietary project, detailed documents and images cannot be shared.
